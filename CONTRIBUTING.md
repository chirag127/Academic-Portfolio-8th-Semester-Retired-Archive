# Contributing to Academic Portfolio

First off, thank you for considering contributing. It's people like you that make this such a great community.

## Where do I go from here?

If you've noticed a bug or have a question, [search the issue tracker](./issues) to see if someone else in the community has already created a ticket. If not, feel free to [create a new issue](./issues/new).

## Fork & create a branch

If you're going to fix a bug or implement a new feature, you'll need to create a fork of the repository and create a new branch.

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
cd academic-portfolio
git checkout -b my-new-feature
```

## Commit your changes

Once you've made your changes, you'll need to commit them to your branch.

```bash
git commit -m "feat: add my new feature"
```

## Push to your fork and submit a pull request

Once you've committed your changes, you'll need to push them to your fork and then submit a pull request.

```bash
git push origin my-new-feature
```

At this point, you should see a link to create a pull request in the GitHub UI. Go ahead and create the pull request.
